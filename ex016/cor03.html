Site para Portfólio

O Projeto

Este pequeno site começa com uma ideia de botar em prática um projeto de desenvolvimento de sistemas web. Mas como tudo tem um começo então este é o começo o site é simples, mas já buscando trabalhar os conceitos de programação e design.

As Ferramentas usadas para desenvolver o site, são ferramentas de uso comum que qualquer pessoa que queira ser programador ou tenha o mínimo de interesse em aprender programação e web developer tem acesso e capacidade de aprender. Basta querer e buscar conhecimento.

Estrutura do Projeto

Este projeto conta com uma estrutura simples utilizando CSS3,HTML5 e JavaScript.Começando com o html que é responsável pela gestão do conteúdo, onde ficam posicionados os parágrafos, títulos e imagens, e botões, formulários de email, fotos e vídeos. E as CSS3, onde implementamos toda a parte de estilos aplicadas em um site, deixando o projeto mais bonito e visível para o cliente, e finalmente o javascript que nos permite fazer interações com a página e automatizando alguns processos.

Minha idéia inicial foi bastante simples, sem muita frescura, apenas botando em prática uma idéia inicial, que já estava em minha mente a muito tempo. Espero que gostem.

Conseito do Projeto

O Principal motivo de trabalhar com estas técnologias é entender o principio de conseito web que fala em principalmente no aspécto da semantica. onde tudo tem um sentido, como se tivesse uma horden para acontecer.

Ao longo deste projeto teremos muitas modificações a fazer, iremos estudar juntos todo o processo de criação de uma página web, aplicando todas as tecnologias capazes de abrilhantar nosso trabalho.Ao longo deste trabalho faremos tudo com HTML,CSS3 e JAVASCRIPT.
